    экспериментальное определение постоянной Стефана-Больцмана при помощи
    оптического пирометра.

    \( S = 2,3\cdot10^{-5}~\text{м}^2; \ \sigma = \cfrac{JU}{S(T^4 - T_0^4)};
    \ T = t + 273 \).
